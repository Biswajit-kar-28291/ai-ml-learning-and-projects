# 🛒 Retail Product & Outlet Analytics Dashboard

## 📌 Project Overview
This project analyzes retail product and outlet data to extract meaningful business insights.  
It helps understand product distribution, pricing patterns, outlet performance, and customer-facing factors like visibility and fat content.

The goal is to simulate a real-world analytics workflow used in companies.

---

## 🎯 Business Problem
Retail companies need to:
- Identify high-value products
- Understand outlet performance
- Optimize product placement
- Analyze pricing trends

This project provides data-driven insights to support these decisions.

## ⚙️ Features

### 🔹 Data Processing
- Data loading from CSV
- Cleaning missing values
- Removing duplicates
- Standardizing columns

### 🔹 Analysis
- Item price (MRP) analysis
- Item type distribution
- Outlet type and size analysis
- Visibility analysis
- Outlet age insights

### 🔹 Visualization
- Item type distribution
- Fat content distribution
- Outlet type distribution
- Location-based analysis
- Average MRP comparisons

### 🔹 Reporting
- KPI summary export
- CSV reports
- Business insights (text file)
